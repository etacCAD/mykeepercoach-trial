## Part 1: Equipment Lists by Age Group

### The "Starter Kit" — What Every Keeper Needs

| Item | U8–U10 | U11–U13 | U14–U16 | U17–U18 | Notes |
|---|---|---|---|---|---|
| **GK Gloves** | ✅ | ✅ | ✅ | ✅ | Properly fitted; replace when grip wears |
| **Soccer Ball (size appropriate)** | Size 3–4 | Size 4–5 | Size 5 | Size 5 | Team usually provides, but own ball for home |
| **GK Jersey (padded elbows)** | Optional | ✅ | ✅ | ✅ | Padded elbows reduce impact fear in beginners |
| **Padded GK Pants / ¾ Tights** | Optional | Recommended | ✅ | ✅ | Hip padding is critical for diving confidence |
| **Knee Pads (compression sleeve)** | Optional | Optional | Optional | Optional | Useful on hard/turf surfaces |
| **Water Bottle** | ✅ | ✅ | ✅ | ✅ | — |

### Diagram 1: Angle String Drill (U10 — Tactical Awareness)

Understanding the "V" of the shooting angle.

```
                        Goal
            [═══════════════════════════]
            |                           |
            |           GK              |
            |        ╱      ╲           |
            |      ╱          ╲         |
            |    ╱              ╲       |
            |  ╱     ANGLE "V"    ╲     |
            |╱                      ╲   |
            ╱                        ╲  |
          ╱                            ╲|
         ●────── BALL (shooter position)

    STRING from left post to ball ━━╱
    STRING from right post to ball ━╲

    GK stands at the NARROWEST point of the V
    = center of the two strings

    MOVE the ball to a new position:

                        Goal
            [═══════════════════════════]
            |                           |
            |                  GK       |  ← GK shifts!
            |                ╱    ╲     |
            |              ╱        ╲   |
            |            ╱            ╲ |
            |          ╱                ╲
            |        ╱                   |
            |      ╱                     |
         ●──────  BALL (wide right)
```

**Key Coaching Point:** When the ball moves wide, GK shifts toward that post. When it's central, GK is more central but further out. The V shape shows WHY.

### Diagram 2: Collapse Dive Progression (U11–U13 — Shot Stopping)

Three phases of dive progression:

```
    PHASE 1: Kneeling

          C (serves ball low to side)
          |
          ⥤ ● →→→→→→
                        GK (kneeling on R knee)
                        ⟿ collapses LEFT

    Landing sequence:
    ┌─────────────────────────────┐
    │  1. THIGH hits ground       │
    │  2. HIP follows             │
    │  3. RIBS/TORSO last         │
    │                             │
    │  Bottom hand: BEHIND ball   │
    │  Top hand: ON TOP of ball   │
    └─────────────────────────────┘

    PHASE 2: Standing

          C
          |
          ⥤ ● →→→→→→
                        GK (standing, ready position)
                        ⟿ collapse dive LEFT

    PHASE 3: With movement

          C
          |
          ⥤ ● →→→→→→
                  ← ← GK (shuffles into position, THEN dives)
                        ⟿ collapse dive LEFT

    ❌ WRONG                    ✅ RIGHT
    ┌──────────────┐            ┌──────────────┐
    │  Diving      │            │  Collapsing   │
    │  FORWARD     │            │  SIDEWAYS     │
    │  (face-first)│            │  (controlled) │
    └──────────────┘            └──────────────┘
```

### Diagram 3: Angle Closing & Save (U11–U13 — Shot Stopping)

```
                           Goal
              [══════════════════════════]
              |            |             |
              |            GK            |
              |                          |
              |                          |
         ○ S3 (15yd)                     |
              |                          |
              |                          |
              |     ○ S2 (12yd)          |
              |                          |
              |                          |
              |            ○ S1 (8yd)    |
              |                          |

    FLOW:
    1. Coach calls "S2!"
    2. GK adjusts angle for S2 position
    3. ● shot comes from S2
    4. GK saves
    5. Coach calls next number

    GK ANGLE POSITIONS:

    From S1 (close, center):    GK is 2–3 yds off line, central
    From S2 (medium, slight R): GK is 1–2 yds off line, shifted right
    From S3 (far, wide left):   GK is on or near line, shifted left

    12 reps total (4 from each position)
```

```
                           Goal
              [══════════════════════════]
              |            |             |
              |            GK            |
              |            |             |
              |            ↓ (advances)  |
              |            |             |
              |            |             |
              |            ↓             |
              |         GK "SET"         |  ← Gets BIG here
              |         position         |
              |            |             |
              |            |             |
              |                          |
              |                          |
              ○────────────○─────────────○  18-yard line
                           |
                           A
                           ↑
                         ● (ball)
                           |

    GK APPROACH SEQUENCE:
    ┌──────────────────────────────────────┐
    │  1. READ: Attacker touches ball      │

```
                              Goal
              [══════════════════════════════]
              |              |               |
              |              GK              |
              |         (starts here)        |
              |                              |
              |        A1         A2         |   ← 2 attackers

              |                              |
              ○──────────────────────────────○  6-yard box

              |                              |

             /                                \
            /                                  \
           /                                    \

    ┌────────────────────────────────────────────┐

    │    └─ Neither?                              │
    │        └─ HOLD position                    │

    └────────────────────────────────────────────┘

```
                           Goal
              [══════════════════════════]
              |            |             |
              |          GK ●            |  GK just made a save
              |            |             |
              |            |             |
              ○────────────○─────────────○  18-yard line
              |            |             |
              |            |             |
              |            |             |
    R1 ×──────|────────────|─────────────|──────× R2
    (25 yd,   |            |             | (25 yd,
     left)    |            |             |  right)
              |            |             |
              |            |             |
              |            |             |
              |            × R3          |
              |         (35 yd,          |
              |          central)        |
              |            |             |
              |            |             |

    GK MUST:
    1. Save the shot  ●
    2. Look up immediately (scan)
    3. Pick a runner (R1, R2, or R3)
    4. Throw (overarm/javelin) to hit runner IN STRIDE
       Arrow: GK ⥤ ●→→→→→→→→ leads the runner →

    TARGET: Hit the runner so they don't have to stop
    12 reps — coach shoots, GK saves & distributes instantly
```

```
                           Goal
              [══════════════════════════]
              |            |             |
              |            GK            |
              |                          |
              ○────────────○─────────────○  6-yard box
              |                          |
              |                          |
              |                          |
              |       S1                 |
              |       (16 yd,            |
              |        central)          |
              |       ⥤ ●               |
              |            ↓ shot 1      |
              ○────────────○─────────────○  18-yard line
             /                            \
            /                              \
           /        S2                      \
          /         (12 yd,                  \
         /           45° angle)               \
                    ⥤ ●
                      ↓ shot 2 (IMMEDIATELY after shot 1)

    SEQUENCE:
    1. S1 shoots from 16 yards → GK saves/parries
    2. IMMEDIATELY (within 2 sec), S2 shoots from angle
    3. GK must recover and save the second shot

    KEY ASSESSMENT:
    ┌────────────────────────────────────────────┐
    │  • Parry DIRECTION on shot 1               │
    │    (away from S2's position = smart parry) │
    │  • Recovery SPEED getting up               │
    │  • Getting SET before shot 2               │
    │  • Composure for second save               │
    └────────────────────────────────────────────┘

    10 rounds — alternate which side S2 is on
```

### Diagram 9: Build From Back Under Press (U14–U16 — Distribution)

```
                           Goal
              [══════════════════════════]
              |            |             |
              |          GK ●            |
              |                          |
              ○────────────○─────────────○  6-yard box
              |                          |
              |    D1 ×         × D2     |  ← Center-backs
              |                          |
              |                          |
    D3 ×      |                          |      × D4
    (LB)      |                          |      (RB)
              |       P1 ×     × P2      |  ← Pressers
              |                          |
              |              × P3        |
              ○────────────○─────────────○  18-yard line

    SCENARIOS:

    Scenario A — Short pass available:
    GK scans → D1 is free → plays to D1 ✅
              P1
         GK ⥤ ● →→ D1 (free)

    Scenario B — Short closed, switch play:
    P1 closes D1 → GK switches to D4 (far side)
              P1 blocks D1
         GK ⥤ ●──────────────────→→ D4 (far side) ✅

    Scenario C — Everything closed, go long:
    All pressers close → GK hits long ball
         GK ⥤ ● ↗↗↗↗↗↗↗↗↗↗↗↗ LONG to midfield ✅

    DECISION HIERARCHY:
    ┌──────────────────────────────────┐
    │  1. Short pass (safest)         │
    │  2. Switch play (next best)     │
    │  3. Long ball (last resort)     │
    │                                 │
    │  NEVER: Panic punt with no aim  │
    └──────────────────────────────────┘

    10 reps — coach tells pressers how aggressively to close
```

### Diagram 10: Sweeper-Keeper Decision Drill (U14–U16 — Tactical)

```
                           Goal
              [══════════════════════════]
              |            |             |
              |            GK            |
              |            |             |
              ○────────────○─────────────○  6-yard box
              |            ↓             |
              |       GK decision:       |
              |    SWEEP or HOLD?        |
              |            |             |
              |       × D (defender)     |
              |                          |
              ○────────────○─────────────○  18-yard line
              |                          |
              |       × A (attacker)     |
              |       running onto       |
              |       through ball       |
              |                          |
              |                          |
    M ⥤ ●→→→→→→→→→→→→→→ through ball
    (midfielder, ~35 yards)

    DECISION FRAMEWORK:
    ┌─────────────────────────────────────────────────┐
    │                                                  │
    │  SWEEP if:                                       │
    │    • Ball is between GK and attacker              │
    │    • GK can reach ball before attacker            │
    │    • Defender can't handle it alone               │
    │                                                  │
    │  HOLD if:                                        │
    │    • Attacker is closer to the ball              │
    │    • Defender has a good angle                   │
    │    • Ball is moving too fast to intercept        │
    │                                                  │
    │  IF SWEEPING:                                    │
    │    • Sprint to ball                              │
    │    • Clear with feet OR pick up (outside box)    │
    │    • Play a pass to D to start a counter         │
    │                                                  │
    │  IF HOLDING:                                     │
    │    • Communicate to D: "YOUR BALL!" or "HOLD!"  │
    │    • Set position for potential 1v1              │
    │                                                  │
    └─────────────────────────────────────────────────┘

    10 reps — coach varies ball speed (slow = sweep, fast = hold)
```
